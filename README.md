# 미결사항
- 잘못된 매개변수(실패 시) : error_occured 출력하는 것(서비스구현 후 추가)
- 개인의 SNS 홈 공간으로 이동할 수 있도록 기능 추가 필요(api 8-e. 문서작성중)
- 소셜 파트 - 개인 컬렉션, 가고싶은 가게
- 랭킹 파트 - 지역별 리뷰어 랭킹 불러오기
- 추가 기능 파트 - 개인 SNS 공간으로 이동하기, 팔로우/워 추가하기
