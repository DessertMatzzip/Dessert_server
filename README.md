#  DssertMatzzip_Server
사용자에게 취향에 맞는 디저트를 추천해주는 어플리케이션 서비스, 기본적으로 사용자간의 SNS서비스를 제공하며 자신의 취향에 맞는 디저트 가게를 리스트로 꾸리거나 별점으로 평가, 혹은 리뷰를 남길 수도 있습니다. 또한 이를 서로 공유할 수 있습니다. 그리고 가게 관리자의 경우 가게 정보를 수정 요청할 수도 있습니다.


### API 문서 : [Google docs로 이동](https://docs.google.com/document/d/1K3fzN_OUAdnAGoy0OsK1ejsdH4eCdTpa25JWxrYZ6W8/edit)


## file structure

- dessertServer : django project

- login : 로그인에 관한 app

- join : 회원 가입에 관한 app

- store : 가게점포에 관한 app

- user : 회원 정보에 관한 app

- login : 회원 로그인을 위한 앱(페이스북, 카카오톡 연동로그인 + 자체로그인)

- join : 회원가입과 회원정보 수정을 위한 앱

- screenshots : 진행상황을 표시하기 위한 이미지 저장 폴더
